# 🎮 Retro Gaming Hub

A static website about classic and handheld retro gaming.
Built as part of a DevOps course lab exercise using Apache HTTP Server.

## Pages

- **Home** — Introduction and navigation
- **Classic Consoles** — From Atari to GBA
- **Top Games** — Titles that shaped gaming history
- **Gaming Tips** — Emulators, save states, and handheld devices

## Tech Stack

- HTML5 & CSS3
- Apache HTTP Server (Ubuntu)
- VirtualHosts with HTTP → HTTPS redirect
- Self-signed SSL certificate (mod_ssl)
- Git & GitHub for version control
